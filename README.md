# Capture-the-Dot

This is a simple capture the dot video game that is made using oled display and a microcontroller(Arduino, ESP-32 etc).

# Components used in the project(can be adjusted as per availability):
  - ESP 32
  - OLED I2C SSD1315 display
  - 4 buttons.
  - A led(can be avoided as per requirment)
  - A passive Buzzer(can be avoided as per requirment)

 
# Libraries required:
 -u8g2 library (by oliver)
